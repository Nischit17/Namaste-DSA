# Number Of Negative Numbers In An Array

Write a function that returns the number of -ve numbers in an array

function countNegatives(arr){
    let count = 0
    for(let i = 0; i < arr.length; i ++){
        if(arr[i] < 0){
            count ++
        }
    }
    return count
}
